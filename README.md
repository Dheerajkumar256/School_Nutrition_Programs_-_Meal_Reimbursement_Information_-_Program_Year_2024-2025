# School_Nutrition_Programs_-_Meal_Reimbursement_Information_-_Program_Year_2024-2025
~ Project Overview
# This project focuses on analyzing School Nutrition Program data to understand meal distribution, eligibility categories, and reimbursement trends across different schools.
# The dashboard provides insights into:

🥳Total meals served (Breakfast & Lunch)
Free vs Paid meal distribution
Monthly trends
Meal type contribution
Reimbursement analysis
Enrollment vs meals served relationship
🎯 Objectives
Analyze meal consumption patterns across schools
Identify the proportion of Free, Paid, and Reduced meals
Study trends in meal distribution over time
Evaluate reimbursement distribution
Understand the relationship between student enrollment and meals served
📌 Key Insights
1. Total Meals Served Analysis
Schools like MILAM EL and CY LAKES HIGH SCHOOL serve the highest number of meals
Lunch contributes more compared to breakfast in most schools
2. Free vs Paid Meal Ratio
67% Free Meals
30% Paid Meals
3% Reduced Meals
➡️ Indicates strong dependency on free meal programs
3. Monthly Trend Analysis
Slight increase in both breakfast and lunch meals from 2024 to 2025
Lunch consistently higher than breakfast
4. Meal Type Contribution
Lunch: 64% (highest contribution)
Breakfast: 35%
Snacks: 1%
5. Reimbursement Analysis
Schools like AUSTIN EL and HOUSTON EL receive higher reimbursements
Lunch reimbursement dominates overall funding
6. Enrollment vs Meals Served
Weak to moderate correlation
Some schools serve more meals than expected enrollment → possible outreach programs
